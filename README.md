# AI Resume Analyzer 📄

A beautiful, glassmorphism-themed Streamlit web application that analyzes PDF resumes, extracts required skills, and scores your profile!

## Features
- 🌈 Smooth animated gradient background
- 🧊 Glassmorphism UI cards with hover effects
- 📊 Automated resume scoring based on missing skills
- 📃 Text extraction directly from PDF resumes

## Setup & Installation

1. Make sure you have **Python** installed.
2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## How to Run

**On Windows:**
Simply double-click the `run.bat` file! It will automatically install any missing dependencies and start the app for you.

**Using Terminal:**
```bash
streamlit run app.py
```
